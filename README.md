## Getting Started

These instructions will help you set up the project on your local machine.

### Prerequisites
```
node@v10.16.0 or higher
npm@6.9.0 or higher
git@2.17.1 or higher
```

### Installation

```bash
# Clone this repository
$ git clone https://github.com/your-username/your-portfolio.git

# Go into the repository
$ cd your-portfolio

# Install dependencies
$ npm install

# Start a local dev server
$ npm start
```

## Customization

1. Update the content in `src/portfolio.js` with your personal information
2. Modify the color scheme in `src/_globalColor.scss`
3. Add your own images and assets in the `src/assets` directory
4. Customize the components in the `src/components` directory

## Deployment

```bash
# Build the website
$ npm run build

# Deploy to GitHub Pages
$ npm run deploy
```
